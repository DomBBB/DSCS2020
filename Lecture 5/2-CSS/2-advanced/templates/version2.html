Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

There are three ways of selecting a html tag

  1. The element selector: selects html elements based on the tag
  2. The Class Selector: Selects elements with a specific class attribute that is set in the tag
  3. The ID selector: They select a html with a specific id attribute that is set in the tag
